10.16.- Renombrando ficheros con rename()



Para cambiar el nombre a un archivo usamos la función rename()



Sinopsis de rename()

#include <stdio.h>

int rename(const char *nombre_antiguo, const char nombre_nuevo);

Esta función devuelve 0 si se ha podido relizar el cambio de nombre o -1 en caso de error


Usaremos esta función más adelante en la eliminación de datos de ficheros, en todo caso se puede ver que la función es muy intuitiva y carece de problemas de entendimiento de su sintaxis.
