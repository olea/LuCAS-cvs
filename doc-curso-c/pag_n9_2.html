9.2.- Función ceil() y floor()                                     



#include <math.h>
double ceil(double valor);

Devuelve el valor entero mayor en formato double



Decimos que ceil() redondea por exceso. Esta función Para compreder un poco esta función, observa el siguiente código:

a = 3.5;

b = -3.5;


printf("a = %f\n b = %f\n", ceil(a), ceil(b));



Que arroja como resultado:


a = 4.000000

b = -3.000000






Funcion floor()



Para el redondeo por defecto usamos la función floor()


#include <math.h>
double floor(double valor);

Devuelve el valor entero menor que double valor en formato double



a = 3.5;

b = -3.5;


printf("a = %f\n b = %f\n", floor(a), floor(b));



Que arroja como resultado:


a = 3.000000

b = -4.000000


La función floor() ha redondeado por defecto los valores de a y b.
