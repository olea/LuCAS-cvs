\maketitle
Responder las siguientes preguntas de manera concisa. Un punto por cada pregunta correcta completa, $1/2$ punto por
cada respuesta correcta incompleta.  

Para aprobar se necesitan 9 puntos.

\begin{enumerate}

\item ?`Dónde se puede encontrar ayuda dentro de un sistema GNU/Linux?

\item ?`A qué se refiere la sigla <<GNU>> en el nombre GNU/Linux?

\item ?`Qué es una distribución? Nombre algunas.

\item ?`Qué tipos de permisos se le pueden asignar a los archivos?

\item ?`Cuáles son las ventajas y desventajas que existen entre un entorno gráfico y un entorno de terminal de texto?

\item Además del precio, ?`qué otra gran ventaja tiene el software libre frente al sofware cerrado?

\item ?`Cómo se representan los dispositivos (periféricos) en GNU/Linux? Ejemplifique.

\item ?`Dónde se guardan las configuraciones del sistema?

\item ?`Qué tipo de archivos y qué importancia tienen los que se encuentran en los directorios \comando{/bin} y \comando{/sbin}?

\item ?`Qué herramientas usaría para comprimir y descomprimir archivos en GNU/Linux?

\item ?`Cuáles son las ventajas de tener un administrador de paquetes como el \comando{rpm} para administrar la instalación/desinstalación de software?

\item ?`Con qué herramientas se pueden cambiar los permisos de archivos?

\item ?`Qué ventajas presenta la organización del sistema de archivos de GNU/Linux?

\item ?`A qué se refiere la filosofía RTFM?

\item ?`Cuáles son las funciones principales que se pueden realizar con los paquetes RPM?

\item ?`Cuál es la licencia por la cual se distribuye la mayoría del software para GNU/Linux, incluyendo este mismo curso? ?`Qué características tiene?

\item ?`Qué navegadores de páginas web se pueden utilizar en GNU/Linux?

\end{enumerate}
